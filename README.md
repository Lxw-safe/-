# 校园智慧物流 - Campus Smart Logistics

## 项目简介

校园智慧物流是一个解决高校快递最后一公里配送难题的Web应用，为师生提供便捷的寄件、取件、配送服务，同时为在校大学生提供勤工助学机会。

## 技术栈

- **框架**: React 18 + Vite
- **UI组件库**: Ant Design 5
- **样式**: Tailwind CSS 3
- **图标**: Ant Design Icons

## 功能模块

### 1. 顶部导航栏 (Navbar)
- Logo和品牌展示
- 导航菜单：首页、我要寄件、我要取件、成为配送员、物流追踪、个人中心
- 登录/注册入口

### 2. 首屏展示区 (Hero Section)
- 吸引人的标题和副标题
- 操作按钮：立即下单、查询物流
- 数据概览卡片
- 校园地图示意图占位

### 3. 智能寄件 (SendPackage)
- 寄件人信息表单（姓名、电话、宿舍楼）
- 收件人信息表单
- 物品类型和重量选择
- 备注信息输入
- 订单提交功能

### 4. 自助取件 (PickupPackage)
- 取件码查询功能
- 扫码取件（模拟）
- 可取包裹列表展示
- 确认取件功能

### 5. 抢单大厅 (DeliveryOrders)
- 待配送订单列表
- 订单详情展示（寄件人、收件人、物品信息）
- 配送费显示
- 抢单功能
- 订单状态管理

### 6. 物流追踪 (Tracking)
- 订单号查询
- 物流时间轴展示
- 配送进度可视化
- 实时状态更新

### 7. 数据看板 (Dashboard)
- 今日订单量
- 活跃配送员人数
- 平均配送时长
- 已完成订单数
- 待处理订单数
- 客户满意度
- 配送效率指标
- 订单分布统计

### 8. 页脚 (Footer)
- 版权信息
- 快速链接
- 关于我们
- 联系方式
- 社交媒体链接

## 安装和运行

### 前置要求
- Node.js 16+ 
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 开发模式运行

```bash
npm run dev
```

应用将在 http://localhost:3000 启动

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 核心业务流程

### 寄件流程
1. 用户填写寄件人和收件人信息
2. 选择物品类型和重量
3. 提交订单
4. 系统生成订单号
5. 配送员抢单
6. 物流配送
7. 签收完成

### 取件流程
1. 用户输入取件码或扫码
2. 系统查询可取包裹
3. 用户确认取件
4. 更新包裹状态为已取件

### 配送流程
1. 学生注册成为配送员
2. 浏览抢单大厅
3. 选择订单并抢单
4. 联系寄件人取件
5. 配送到收件人
6. 确认签收
7. 获得配送费
